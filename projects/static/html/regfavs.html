I wrote this program because RegEdit's favorites feature is, in my opinion, too limited. RegEdit favorites are displayed in the order they are created and are not moveable or sortable by default. If you're like me, and have quite a few favorites, it's nice to keep them in some kind of order. It's makes navigating the favorites menu a little bit quicker. RegFavs is not a full registry editor, it tricks RegEdit into thinking you saved your favorites in the chosen order, therefor allowing you to organize them how ever you like. You can sort by ascending/descending alphabetical order, or just move things around any way you see fit. RegFavs has never lost my favorites, or corrupted anything, but it's always a good idea for you to backup your favorites using RegEdit or RegFavs before you start editing. You can use the Export/Import functionality of either application to do so.
All features are self-explanatory, all keyboard shortcuts are shown on the menu.

Contact me at cj@welbornprod.com if you run into any problems. If you've never installed a Visual Basic .Net application before, you might need the runtime libraries. They're easily installed from microsoft.com.
